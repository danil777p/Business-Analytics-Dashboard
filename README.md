# 📊 Business Analytics Dashboard

A Streamlit-based dashboard for analyzing organizations, employees, and project performance using SQL and AI-powered insights.

---

## 🚀 Features

* 📈 Interactive visualizations using Plotly
* 🧠 AI-generated insights from data
* 🗄️ MySQL database integration
* 📊 Data processing with Pandas
* ⚡ Fast and user-friendly interface with Streamlit

---

## 🛠️ Tech Stack

* Python
* Streamlit
* MySQL
* Pandas
* Plotly
* SQLAlchemy

---

## ⚙️ Installation

Clone the repository:

git clone https://github.com/dani777p/Business-Analytics-Dashboard.git
cd Business-Analytics-Dashboard

Install dependencies:

pip install -r requirements.txt

---

## 🔑 Environment Setup

Create a `.env` file in the root folder and add:

DB_USER=your_user
DB_PASS=your_password
DB_HOST=127.0.0.1
DB_NAME=for_dashboard

---

## ▶️ Run the Application

streamlit run dashboard.py
